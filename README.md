# vue-template-fy

#### 介绍
PC端模板
主要用于后台管理平台、可视化大数据(集成ehcarts)

#### 软件架构
根据vue-admin-template修改而来；


#### 安装教程

# clone the project
git clone https://github.com/PanJiaChen/vue-element-admin.git

# enter the project directory
cd vue-element-admin

# install dependency
npm install

# develop
npm run dev
